# Website-project
# Single Page – Two CSS Versions

#File Organization
- `index.html` – The main HTML file used for both versions (A and B).  
- `styleA.css` – Styles for Version A (vertical layout with 6 boxes).  
- `styleB.css` – Styles for Version B (horizontal layout + fixed bottom-right element).  

To switch between versions, change:
```html
<link rel="stylesheet" href="styleA.css">
To
<link rel="stylesheet" href="styleB.css">
